# Final C++ Project

This is my final project for C++.  
The program is a console application that allows the user to manage and work with data through a simple menu.

## Features
- Add data
- View data
- Update data
- Delete data
- Perform some calculations

## How to Run

1. Clone the repository:
   git clone https://github.com/engdoaagamal/final-project-c-.git

2. Open the project folder:
   cd final-project-c-

3. Compile the code:
   g++ main.cpp -o program

4. Run the program:
   ./program

## Usage
When you run the program, a menu will appear.  
Choose the operation by entering the number, then follow the instructions on the screen.

## Notes
- The project is written in C++
- Uses basic concepts like functions, arrays, and classes
